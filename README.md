# RentaRide

A MERN stack vehicle rental system.

## Features
- User authentication (JWT)
- Vehicle listing
- Booking system
- Booking approval/rejection
- Prevents double booking

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
